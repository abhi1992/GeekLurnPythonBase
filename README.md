# GeekLurnPythonBase
A base website with flask and bootstrap to manipulate svgs.

## Installation
```
> git clone https://github.com/abhi1992/GeekLurnPythonBase.git
> cd .\GeekLurnPythonBase\
> py -3 -m venv venv
> venv\Scripts\activate
> pip3 install flask
> flask --app src/index.py --debug run
```
